Persistent Reasoning Engine

An evolving enterprise knowledge model over time. The output is a structured enterprise model with Doctrine, Capabilities, Active States, Active Obligations, Risks, Management Decisions, Causal Relationships, and Enterprise Trajectory — all with full provenance and temporal validity.

API Endpoints

GET /health — service health check 
curl http://localhost:8000/health
# {"status": "ok"}
GET /entities/{ticker}/state — currently-valid facts 
curl http://localhost:8000/entities/MSFT/state | jq
# {
#   "ticker": "MSFT",
#   "state": [
#     {"category": "doctrine", "fact_json": {...}, "valid_from": "...", ...},
#     {"category": "capability", ...}
#   ]
# }
GET /entities/{ticker}/state-as-of?timestamp=... — bitemporal query 
curl 'http://localhost:8000/entities/MSFT/state-as-of?timestamp=2023-06-01T00:00:00Z' | jq
GET /entities/{ticker}/changes?from=...&to=... — temporal diff 
curl 'http://localhost:8000/entities/MSFT/changes?from=2022-01-01T00:00:00Z&to=2024-12-31T00:00:00Z' | jq
GET /entities/{ticker}/risks-active — currently-active risks 
curl http://localhost:8000/entities/MSFT/risks-active | jq
